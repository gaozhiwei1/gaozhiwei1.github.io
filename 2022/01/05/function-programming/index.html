1. 过滤并结构化数据

我们先上一段代码展示一下

from typing import Iterator, Tuple, Callable

def Group_filter_iter(n:int, pred: Callable, items: Iterator)->Iterator:
    subset = filter(pred, items)
    row = tuple(next(subset) for i in range(n))
    while row:
        yield row
        row = tuple(next(subset) for i in range(n))

if __name__ == "__main__":
    Iter = Group_filter_iter(7, lambda x:x%3 ==0 or x%5==0, range(1, 100))
    print(next(Iter))
    print(next(Iter))

首先从typing中导入一些参数的类型，我们产生一个分组的过滤器，Callable其实就是一个函数，输入类型为随意，输出为any，这时我们调用filter函数产生一个，这时候我们就需要生成一个迭代器。我们最终得到下面的迭代器结果。

(3, 5, 6, 9, 10, 12, 15)
(18, 20, 21, 24, 25, 27, 30)

2. 使用可调用对象构造高阶函数

这里我们以一个类作为简单的说明

from typing import Callable, Optional, Any
import math

class NullAware:
    def __init__(self, some_func: Callable[[Any], Any]) -> None:
        self.some_func = some_func
    
    def __call__(self, arg: Optional[Any]) -> Optional[Any]:
        return None if arg is None else self.some_func(arg)

这里我们重载了类的__call__函数，使之可以适应输入值为None的情况，同样也将输出值设为None。最终我们得到结果如下

null_log_scale = NullAware(math.log)
list(map(null_log_scale, [1,2,3,4,None]))

3. 递归与归约

这里我们先介绍一下泛型类型

from typing import TypeVar, Generic
from logging import Logger

T = TypeVar('T')

class LoggedVar(Generic[T]):
    def __init__(self, value: T, name: str, logger: Logger) -> None:
        self.name = name
        self.logger = logger
        self.value = value

    def set(self, new: T) -> None:
        self.log('Set ' + repr(self.value))
        self.value = new

    def get(self) -> T:
        self.log('Get ' + repr(self.value))
        return self.value

    def log(self, message: str) -> None:
        self.logger.info('%s: %s', self.name, message)

实际上这里的TypeVar和Optional有很大的共同点，也就是说TypeVar可以按照自己的名字来构建，是比较自由的类型，很容易记住。实际上任何的函数都可以转化为递归函数，在此简单的介绍一下乘积函数的递归形式。

\begin{equation*} prod(C) = \left\{\begin{array}{c} 1\quad\quad\quad\quad\quad\quad\quad\quad\quad len(C) = 0\\ C[0]\times prod(C[1:])\quad len(C)> 0 \end{array} \right. \end{equation*}

这里对象如果是列表的话，是最简单的形式，代码如下

def prodrc(collection: Sequence[float])->float:
    if len(collection) == 0:
        return 1
    return collection[0]*prodrc(collection[1:])

如果参数是可迭代对象的话，代码如下

def prodri(items: Iterator[float])->float:
    try:
        head = next(items)
    except StopIteration:
        return 1
    return head * next(items)

这里我们充分利用到了可迭代对象的一个优势，那便是可以这样进行迭代，但是此处如果没有进行尾调用优化的话，效率可能并没有那么高。

first_leg = (

from typing import NamedTuple

在这种情况下我们进行数据构造，可以看到结构非常的清晰明显， 可读性很强

6. Itertools模块

这里我们首先要强调一下可迭代对象的一个重要缺陷:只能使用一次

6.1 使用count()进行计数

enumerate = lambda x, start = 0: zip(count(start), x)

这里其实就是构建一个处理迭代器的对象，zip可以处理迭代器，其本身生成的也是迭代器。

6.2 使用实数参数计数

until函数定义为从一个迭代器中不断取值，直到满足终止条件

from itertools import count
Generator = Iterator[Tuple[float, float]]
source:Generator = zip(count(0,0.1), (0.1*c for c in count()))

Extractor = Callable[[Tuple[float, float]], float]
x:Extractor = lambda x_y:x_y[0]
y:Extractor = lambda x_y:x_y[1]

Comparator = Callable[[Tuple[float, float]], bool]
neq: Comparator = lambda xy: abs(x(xy) - y(xy))>1e-12

这里我们用count函数生成一个无限迭代器，计算下面两个计算的细微差异 ∑x∈N0.1\sum_{x\in N}0.1∑x∈N​0.1和0.1∗∑x∈N10.1*\sum_{x\in N}10.1∗∑x∈N​1。结果发现相当显著。

6.3 用cycle()循环迭代

可以用cycle()生成一系列的逻辑判断值，例如如下

m3 = (i == 0 for i in cycle(range(3)))
m5 = (i == 0 for i in cycle(range(5)))

这样就可以生成一系列标识值如下

multipliers = zip(range(10), m3, m5)

然后我们过滤数值可以得到

total = sum(i
    for i, * multipliers in multipliers
    if any(multipliers))

这里的*求完值之后就会变成一个列表就行计算。这样才能用any函数进行计算。这里我们使用cycle函数就能很简单的从大量的数据集中挑选出少量的数据集进行一些验证，极大的减少了我们的工作量。

6.4 用repeat()函数进行重复取值

all = repeat(0)
subset = cycle(range(100))
def randseq(limit):
    while True:
        yield random.randrange(limit)
randomlized = randseq(100)
choose = lambda rule: (x == 0 for x in rule)

实际上是用来选取所有的当前值的函数，这与cycle有选择地进行选值还是有很大的区别。这里我们简单的使用这三种方式，就可以随意的进行整个挑选，部分挑选还有随机挑选，实际上是很有用处的。

7. 使用有限迭代器

itertools模块包含很多有用的函数，现在将其总结如下

  • enumerate():实际上该函数在_builtins_包中，但可应用于迭代器，且用法与itertools模块中函数类似
  • accumulate():返回作为输入的可迭代对象的归约序列，他是高阶函数。
  • chain(): 将多个可迭代对象按照顺序组合在一起.
  • groupby(): 将输入的可迭代对象按照指定函数分割为多个子迭代对象。
  • zip_longest(): 同zip函数，但是会进行短序列补充。
  • islice(): 序列切片函数的可迭代对象版本。
  • dropwhile() 与 takewhile(): 使用bool函数过滤可迭代对象。
  • compress():根据第二个布尔可迭代对象形式的输入参数对第一个可迭代对象进行筛选。
  • filterfalse():对可迭代对象应用过滤函数。
  • starmap(): 将函数用于右元组组成的可迭代序列。

filterfalse()函数用法

Python filterfalse()和filter()过滤方法，前面用到了filter()函数，可以基于它定义itertools模块中的filterfalse()函数，如下所示：

filterfalse = (lambda pred, iterable:
    filter(lambda x: not pred(x), iterable)
)

基于filter()函数定义意味着谓词函数可以是None。函数filter(None, iterable)返回可迭代对象中的所有真值，函数filterfalse(None, iterable)则返回所有非真值。

>>> filter(None, [0, False, 1, 2])
>>> list(_)
[1, 2]

>>> filterfalse(None, [0, False, 1, 2])
<itertools.filterfalse object at 0x101b43a50>
>>> list(_)
[0, False]

使用filterfalse()函数意在提高代码复用性。使用这个简洁的函数，可以方便地将输入数据分为“通过”和“舍弃”两组，而无须使用逻辑非这种繁复的表达方式。

该思想体现如下：

iter_1, iter_2 = iter(some_source), iter(some_source)
good = filter(test, iter_1)
bad = filterfalse(test, iter_2)

starmap()函数

itertools模块的starmap()函数实际上是map()函数的*a版本，如下定义：

starmap = (lambda function, arg_iter:
    (function(*a) for a in arg_iter)
)
from itertools import starmap 
  
li =[(2, 5), (3, 2), (4, 3)] 
  
new_li = list(starmap(pow, li)) 
  
print(new_li)
[32,9,64]
